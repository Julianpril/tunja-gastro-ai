# Tunja Gastro AI

Monorepo for the Tunja Gastro AI project.

## Quick Start (Windows)

Para iniciar todo el proyecto (Backend + Frontend) con un solo comando:

1. Abre PowerShell en la carpeta raíz.
2. Ejecuta:
   ```powershell
   .\start_dev.ps1
   ```

Esto abrirá dos ventanas nuevas:
- Una para el **Backend** (FastAPI) en http://localhost:8000
- Otra para el **Frontend** (Expo) donde podrás escanear el código QR.

## Structure
- `app/`: React Native Frontend
- `backend/`: FastAPI Backend
- `ml/`: Machine Learning Pipeline
- `docs/`: Documentation
